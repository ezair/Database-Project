Software used

The Website

Bootstrap was used for the index.html page to create a smooth template. For the backend/login page, Flask was used with the flask-bootstrap library.

The Game

Unreal Engine 4 was used for the creation of the samle video game. Additional software like blender was used for 3D modeling. A plugin was used for database connection.

The Database

For the database SQL Server 2017 was used. To connect the website to the database we used Pyodbc inside of the main.py script, which contains the code for the login page.
